We welcome and appreciate contributions of all kinds to the Activepieces project. Whether you are a developer, designer, user, or simply have suggestions for improvements, we encourage you to get involved.

## Building Pieces
We encourage you to build new [Pieces](../pieces/overview) as they are the main driver for usage growth. If you'd like to build a piece, check out [Building Pieces](./building-pieces/quickstart) for instructions and best practices.

## How to Contribute

Here are some ways you can contribute to the Activepieces project:

- Report bugs or request new features by opening an issue on the [GitHub issue tracker](https://github.com/activepieces/activepieces/issues).
- Suggest improvements or report documentation issues by opening an issue on the [GitHub issue tracker](https://github.com/activepieces/activepieces/issues).

## Code of Conduct

The Activepieces project has a [Code of Conduct](https://github.com/activepieces/activepieces/blob/main/.github/CODE_OF_CONDUCT.md) that we ask all contributors to follow. Please make sure to read and understand it before contributing.

Thanks for considering contributing! We look forward to working with you.
