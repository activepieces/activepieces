## Actions


      <Card title="Send Notification " >
        Send a notification to ntfy

        <Expandable title="Properties">
          
<ResponseField name="Topic" required type="Short_text">
The topic/channel to send the notification to, e.g. test1
</ResponseField>
            
<ResponseField name="Title"  type="Short_text">
The title of the notification
</ResponseField>
            
<ResponseField name="Message" required type="Long_text">
The message to send
</ResponseField>
            
<ResponseField name="Priority"  type="Short_text">
The priority of the notification (1-5). 1 is lowest priority.
</ResponseField>
            
<ResponseField name="Tags"  type="Array">
The tags for the notification.
</ResponseField>
            
<ResponseField name="Icon"  type="Short_text">
The absolute URL to your icon, e.g. `https://example.com/communityIcon_xnt6chtnr2j21.png`
</ResponseField>
            
<ResponseField name="Actions"  type="Long_text">
Add Action buttons to notifications, see `https://docs.ntfy.sh/publish/#action-buttons`
</ResponseField>
            
<ResponseField name="Click"  type="Short_text">
You can define which URL to open when a notification is clicked, see `https://docs.ntfy.sh/publish/#click-action`
</ResponseField>
            
<ResponseField name="Delay"  type="Short_text">
Let ntfy send messages at a later date, e.g. 'tomorrow, 10am', see `https://docs.ntfy.sh/publish/#scheduled-delivery`
</ResponseField>
              
        </Expandable>
      </Card>
