# ClayStack

ClayStack is a decentralized liquid staking protocol that enables you to stake MATIC/POL on Polygon PoS and receive csMATIC — a liquid staking derivative you can use across the DeFi ecosystem.

## Actions

- **Get Protocol TVL** - Fetch the current Total Value Locked (TVL) from DeFiLlama with 1h/1d/7d percentage changes
- **Get CS Token Price** - Fetch ClayStack CS token price, market cap, and 24h change from CoinGecko
- **Get Chain Breakdown** - Fetch TVL distribution across chains, sorted by size, with percentage of total
- **Get TVL History** - Fetch historical TVL data over a configurable number of days with % change from baseline
- **Get Protocol Stats** - Combined stats fetching TVL and CS price data in parallel

## Authentication

No authentication required — all data is fetched from public APIs (DeFiLlama, CoinGecko).
