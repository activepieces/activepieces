# Balancer

Balancer is a decentralized automated market maker (AMM) protocol built on Ethereum that allows for customizable liquidity pools with weighted token ratios. Unlike traditional 50/50 AMMs, Balancer supports pools with up to 8 tokens at custom weights. BAL is the governance token.

## Actions

- **Get Protocol TVL** — Current TVL from DeFiLlama with 1h/1d/7d changes
- **Get BAL Token Price** — Live BAL price, market cap, 24h volume from CoinGecko
- **Get Chain TVL Breakdown** — TVL by chain (Ethereum, Arbitrum, Polygon, etc.)
- **Get TVL History** — Historical TVL data points
- **Get Protocol Stats** — Comprehensive protocol + token stats summary
