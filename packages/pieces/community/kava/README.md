# Kava

Kava is a Layer-1 blockchain and DeFi platform that combines the Ethereum and Cosmos ecosystems. 
It offers lending, USDX stablecoin minting, and a decentralized exchange (DEX).
KAVA is the native governance and staking token.

## Actions

- **Get Protocol TVL** - Fetch total value locked in the Kava protocol via DeFiLlama
- **Get KAVA Price** - Get current KAVA token price from CoinGecko
- **Get Chain Breakdown** - Get TVL breakdown by chain for Kava via DeFiLlama
- **Get TVL History** - Retrieve the last 30 days of historical TVL for Kava
- **Get Protocol Stats** - Get key protocol statistics (TVL, chains, category) from DeFiLlama

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) — Free, no authentication required
- [CoinGecko API](https://www.coingecko.com/en/api) — Free, no authentication required
