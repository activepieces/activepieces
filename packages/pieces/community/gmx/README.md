# GMX Piece

This piece provides actions to fetch data from the GMX decentralized perpetual exchange using DeFiLlama and CoinGecko APIs.

## Actions

| Action | Description |
|--------|-------------|
| Get Protocol TVL | Fetch GMX total value locked with 1h/1d/7d changes |
| Get GMX Token Price | Fetch live GMX token price, market cap, and 24h volume |
| Get Chain TVL Breakdown | Get TVL distribution across chains |
| Get TVL History | Retrieve historical TVL data points |
| Get Protocol Stats | Combined protocol and token statistics summary |

## Data Sources

- [DeFiLlama](https://defillama.com/protocol/gmx)
- [CoinGecko](https://www.coingecko.com/en/coins/gmx)

No authentication required.
