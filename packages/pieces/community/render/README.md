# Render Piece

Manage your Render cloud services via the Render API.

## Actions

| Action | Description |
|--------|-------------|
| List Services | List all services in your Render account |
| Get Service | Get details for a specific service |
| Deploy Service | Trigger a new deploy for a service |
| List Deploys | List recent deploys for a service |
| Get Deploy | Get details for a specific deploy |

## Authentication

Requires a Render API key. Generate one at https://dashboard.render.com/u/settings#api-keys
