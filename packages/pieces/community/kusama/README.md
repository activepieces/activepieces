# Kusama (KSM)

Kusama is Polkadot's canary network — a fast-moving, experimental blockchain for deploying and testing new features before Polkadot. KSM is the native token.

## Actions

- **Get Protocol TVL** — Fetch current total value locked for Kusama from DeFiLlama
- **Get KSM Price** — Get current KSM price, market cap, and 24h change from CoinGecko
- **Get Chain Breakdown** — Retrieve TVL breakdown by chain from DeFiLlama
- **Get TVL History** — Fetch last 30 days of historical TVL data from DeFiLlama
- **Get Protocol Stats** — Get key protocol statistics (TVL, chains, category) from DeFiLlama

## APIs Used

- [DeFiLlama](https://defillama.com/docs/api) — Free, no authentication required
- [CoinGecko](https://www.coingecko.com/en/api) — Free public API, no authentication required
