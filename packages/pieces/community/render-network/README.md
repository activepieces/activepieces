# Render Network

Render Network is a decentralized GPU compute network that connects artists and studios needing GPU power with node operators who supply it. Originally built on Ethereum as RNDR, the network migrated to Solana and rebranded to RENDER token.

This piece provides actions to monitor Render Network token price, market statistics, network overview, historical prices, and top token holders — all via free public APIs (no API key required).
