# Acala Network

Acala is a DeFi hub and Layer-1 parachain on Polkadot, offering a stablecoin (aUSD), liquid staking (LDOT), and an AMM DEX. ACA is the governance token.

## Actions

- **Get Protocol TVL** – Fetch current Total Value Locked for Acala via DeFiLlama
- **Get ACA Price** – Fetch current ACA token price and market data via CoinGecko
- **Get Chain Breakdown** – Fetch TVL breakdown by chain for Acala via DeFiLlama
- **Get TVL History** – Fetch last 30 days of historical TVL data for Acala via DeFiLlama
- **Get Protocol Stats** – Fetch key protocol statistics (TVL, chains, category) for Acala via DeFiLlama

All APIs are free and require no authentication.
