# Multichain (formerly AnySwap)

Multichain is a cross-chain router protocol supporting 90+ blockchains. MULTI is the governance token.

## Actions

- **Get Protocol TVL** — Current total value locked via DeFiLlama
- **Get MULTI Price** — MULTI governance token price via CoinGecko
- **Get Chain Breakdown** — TVL breakdown by blockchain
- **Get TVL History** — Last 30 days of historical TVL
- **Get Protocol Stats** — Key stats: TVL, chains, category

## APIs

- [DeFiLlama](https://defillama.com/docs/api) — Free, no auth
- [CoinGecko](https://www.coingecko.com/en/api) — Free, no auth
