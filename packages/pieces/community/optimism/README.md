# Optimism Piece for Activepieces

Connects to Optimism Layer-2 rollup data via free public APIs (DeFiLlama and CoinGecko). No API key required.

## Actions

- **Get Protocol TVL** - Fetch current Optimism total value locked from DeFiLlama
- **Get OP Price** - Get OP governance token price and market data from CoinGecko
- **Get Chain Breakdown** - TVL breakdown by chain from DeFiLlama
- **Get TVL History** - Last 30 days of historical TVL data
- **Get Protocol Stats** - Key stats: TVL, chains, category, and metadata

## APIs Used

- [DeFiLlama](https://api.llama.fi) - Free, no authentication required
- [CoinGecko](https://api.coingecko.com) - Free tier, no authentication required
