# Bifrost Finance

Bifrost is a liquid staking protocol in the Polkadot/Kusama ecosystem.
It offers vTokens (vDOT, vKSM, vBNB, vETH) as liquid staking derivatives.
BNC is the governance token.

## Actions

- **Get Protocol TVL**: Fetch current Total Value Locked for Bifrost Finance via DeFiLlama.
- **Get BNC Price**: Fetch current BNC token price and market data via CoinGecko.
- **Get Chain Breakdown**: Fetch TVL breakdown by blockchain via DeFiLlama.
- **Get TVL History**: Fetch 30-day historical TVL data via DeFiLlama.
- **Get Protocol Stats**: Fetch key protocol statistics (TVL, chains, category) via DeFiLlama.

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) — free, no auth required
- [CoinGecko API](https://www.coingecko.com/en/api) — free tier, no auth required
