# Compound Finance

Compound Finance is the original DeFi lending protocol (2018) and pioneer of liquidity mining. COMP token enables governance. Compound V3 (Comet) offers isolated collateral markets with improved capital efficiency.

## Actions

- **Get Protocol TVL** — Fetches current Total Value Locked with 1h/1d/7d changes from DeFiLlama
- **Get COMP Token Price** — Fetches live COMP price, market cap, and 24h volume from CoinGecko
- **Get Chain TVL Breakdown** — Returns TVL distribution across Ethereum, Arbitrum, Polygon, Base, and other chains
- **Get TVL History** — Returns configurable historical TVL data points
- **Get Protocol Stats** — Returns a combined summary of protocol metrics and COMP token data

## Data Sources

- [DeFiLlama API](https://defillama.com/docs/api) (no authentication required)
- [CoinGecko API](https://www.coingecko.com/en/api) (no authentication required)

## Authentication

No authentication required. This piece uses public APIs.
