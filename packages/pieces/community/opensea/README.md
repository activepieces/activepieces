# OpenSea

OpenSea is the world's largest NFT marketplace. This piece integrates with the OpenSea API v2 to provide NFT data, collection stats, listings, and offers.

## Authentication

You need an OpenSea API key. Get one at [docs.opensea.io/reference/api-keys](https://docs.opensea.io/reference/api-keys).

## Actions

- **Get NFT** - Get metadata and details for a single NFT by contract address and token ID
- **List NFTs by Collection** - List NFTs within a specific collection by collection slug
- **Get Collection Stats** - Get floor price, total volume, number of owners, and other statistics for a collection
- **Get Best Listings** - Get the best (lowest price) active listings for a collection
- **Get Best Offers** - Get the best (highest value) active offers for a collection
