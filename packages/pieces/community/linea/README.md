# @activepieces/piece-linea

Linea zkEVM Layer-2 analytics piece for Activepieces.

Fetches TVL, ETH price, chain breakdowns, and historical data via free public APIs (DeFiLlama + CoinGecko). No API key required.

## Actions

- **Get Protocol TVL** — Current TVL from DeFiLlama
- **Get ETH Price** — Current ETH price from CoinGecko
- **Get Chain Breakdown** — TVL breakdown by chain from DeFiLlama
- **Get TVL History** — Last 30 days historical TVL from DeFiLlama
- **Get Protocol Stats** — Key stats (TVL, chains, category) from DeFiLlama
