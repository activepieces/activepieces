# Aerodrome Finance — Activepieces Piece

> **The leading DEX on Base chain** — ve(3,3) tokenomics, fork of Velodrome.

This piece connects Activepieces to Aerodrome Finance public DeFi APIs (no auth required).

## Actions

| Action | Description | API |
|--------|-------------|-----|
| **Get Protocol TVL** | Total Value Locked across all Aerodrome pools | DeFiLlama |
| **Get AERO Price** | Current price, market cap & 24 hr volume | CoinGecko |
| **Get Top Pools** | Top liquidity pools on Aerodrome v2 | DeFiLlama Yields |
| **Get Pool APYs** | Pool APYs with optional minimum APY filter | DeFiLlama Yields |
| **Get Chain Stats** | Base chain stats (TVL, volume, etc.) | DeFiLlama |

## Auth

None — all APIs are free and public.
