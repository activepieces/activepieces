# Alpaca Finance

Alpaca Finance is the largest leveraged yield farming protocol on BNB Chain. It allows users to lend assets to earn yield and borrow assets to farm with leverage up to 6x. ALPACA is the governance token.

## Actions

- **Get Protocol TVL** - Fetch current TVL data for Alpaca Finance from DeFiLlama
- **Get ALPACA Price** - Fetch current ALPACA token price data from CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by blockchain from DeFiLlama
- **Get TVL History** - Fetch last 30 days of historical TVL data from DeFiLlama
- **Get Protocol Stats** - Fetch key protocol statistics (TVL, chain count, category) from DeFiLlama

## APIs Used

- [DeFiLlama](https://defillama.com/docs/api) - Free, no authentication required
- [CoinGecko](https://www.coingecko.com/en/api/documentation) - Free tier, no authentication required
