# Across Protocol

Across Protocol is a cross-chain bridge optimized for speed and low fees. It uses UMA's optimistic oracle for security and enables fast token transfers between EVM-compatible networks. ACX is the governance token of the Across ecosystem.

## Actions

| Action | Description |
|--------|-------------|
| **Get Protocol TVL** | Fetch total value locked across all chains via DeFiLlama |
| **Get ACX Price** | Get current ACX token price and market data via CoinGecko |
| **Get Chain Breakdown** | Get TVL distribution across all supported blockchains |
| **Get TVL History** | Retrieve last 30 days of historical TVL data |
| **Get Protocol Stats** | Key stats: TVL, chain count, 7d change, category |

## APIs

- **[DeFiLlama API](https://defillama.com/docs/api)** — Free, public, no authentication required
- **[CoinGecko API](https://www.coingecko.com/en/api)** — Free tier, no API key required

## Authentication

No API key required. All data is sourced from free public APIs.
