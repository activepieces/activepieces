# Moonriver

Moonriver is an EVM-compatible Kusama parachain — the canary network to Moonbeam on Polkadot. MOVR is the native token.

## Actions

- **Get Protocol TVL** - Fetch total value locked on Moonriver via DeFiLlama
- **Get MOVR Price** - Fetch current MOVR token price via CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by blockchain via DeFiLlama
- **Get TVL History** - Fetch last 30 days of historical TVL data via DeFiLlama
- **Get Protocol Stats** - Fetch key protocol statistics (TVL, chains, category) via DeFiLlama

All APIs are free and require no authentication.
