# PDF Toolkit Piece

This piece provides a collection of essential PDF manipulation actions that run fully in-memory without external services.

## Features

- **Merge PDF** — combine multiple PDF files into a single document  
- **Split PDF** — extract selected page ranges  
- **Extract Text** — read plain text from PDF content  
- **Extract Pages** — export specific pages as standalone PDFs  
- **Compress PDF** — reduce file size  
- **Add Watermark** — overlay text watermark on every page  

## Compatibility
Fully compatible with the Activepieces framework and APFile format.

## Notes
All operations use `pdf-lib` and are safe for serverless runtime environments.
