# Nano (XNO)

Activepieces piece for [Nano](https://nano.org) — a feeless, instant digital currency using a block-lattice data structure. XNO is the native token.

## Actions

- **Get Protocol TVL** — Fetch current Total Value Locked for Nano from DeFiLlama
- **Get Token Price** — Fetch current XNO price from CoinGecko
- **Get Chain Breakdown** — Fetch Nano TVL broken down by chain from DeFiLlama
- **Get TVL History** — Fetch Nano TVL history for the last 30 days from DeFiLlama
- **Get Protocol Stats** — Fetch key protocol statistics for Nano from DeFiLlama

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) — No API key required
- [CoinGecko API](https://www.coingecko.com/en/api) — No API key required
