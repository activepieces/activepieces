# MultiversX (EGLD) Piece

Activepieces community piece for **MultiversX** (formerly Elrond) — a high-throughput Layer-1 blockchain using Adaptive State Sharding and Secure Proof of Stake. EGLD (eGold) is the native token.

## Actions

- **Get Protocol TVL** — Fetch current Total Value Locked for MultiversX via DeFiLlama
- **Get EGLD Price** — Fetch current EGLD price, market cap, and 24h change via CoinGecko
- **Get Chain Breakdown** — Get TVL breakdown by chain (MultiversX vs bridged assets) via DeFiLlama
- **Get TVL History** — Fetch last 30 days of historical TVL data via DeFiLlama
- **Get Protocol Stats** — Get key protocol statistics (TVL, chains, category) via DeFiLlama

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) — Free, no authentication required
- [CoinGecko API](https://www.coingecko.com/en/api) — Free tier, no authentication required
