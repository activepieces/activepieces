# Origin Ether

Origin Ether (OETH) is a yield-bearing ETH liquid staking token by Origin Protocol. OETH automatically earns yield from multiple DeFi strategies, including Curve, Convex, and native ETH staking.

## Actions

### Get Protocol TVL
Fetches the current Total Value Locked (TVL) for the Origin Ether protocol from DeFiLlama.

### Get OETH Price
Fetches the current OETH token price, market capitalization, and 24-hour price change from CoinGecko.

### Get Chain Breakdown
Returns a breakdown of TVL by blockchain network, sorted by TVL descending, with each chain's percentage of total TVL.

### Get TVL History
Returns historical TVL data for a configurable number of days (default: 30), including the percentage change from the baseline (first data point).

### Get Protocol Stats
Fetches a combined summary of protocol TVL and OETH price data in a single parallel API call — ideal for dashboards and monitoring.

## Authentication

No authentication required. All data is fetched from public APIs (DeFiLlama and CoinGecko).
