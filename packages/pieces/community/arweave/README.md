# Arweave Piece

Fetch data from the Arweave permanent storage network using DeFiLlama and CoinGecko APIs.

## Actions

| Action | Description |
|--------|-------------|
| Get Protocol TVL | Arweave TVL with 1h/1d/7d changes |
| Get AR Token Price | Live AR price, market cap, 24h volume |
| Get Chain TVL Breakdown | Top protocols built on Arweave |
| Get TVL History | Historical TVL data points |
| Get Protocol Stats | Combined protocol + token summary |

## Data Sources
- [DeFiLlama](https://defillama.com/chain/Arweave)
- [CoinGecko](https://www.coingecko.com/en/coins/arweave)

No authentication required.
