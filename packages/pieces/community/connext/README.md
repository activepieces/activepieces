# Connext

Connext is a cross-chain liquidity network enabling fast, trust-minimized token transfers across EVM chains. NEXT is the governance token.

## Actions

- **Get Protocol TVL** - Fetch the current total value locked in Connext protocol
- **Get NEXT Token Price** - Get the current price of NEXT governance token via CoinGecko
- **Get Chain Breakdown** - Get TVL breakdown across all supported chains
- **Get TVL History** - Retrieve last 30 days of historical TVL data
- **Get Protocol Stats** - Get key protocol statistics (TVL, chains count, category)

## APIs Used

- [DeFiLlama](https://defillama.com/docs/api) - Free, no authentication required
- [CoinGecko](https://www.coingecko.com/en/api) - Free tier, no authentication required
