# CoinMetrics

Institutional-grade crypto network data and analytics via the CoinMetrics Community API.

## Features

- **No API key required** — uses the free CoinMetrics Community API
- Time series metrics for assets (price, volume, market cap, active addresses, etc.)
- Asset and metrics catalog
- Trading pair metrics
- Exchange-level metrics

## Actions

- **Get Asset Metrics** — Fetch time series metrics for a specific crypto asset (BTC, ETH, etc.)
- **Get Asset Catalog** — List all supported assets with IDs and names
- **Get Metrics Catalog** — List all available metrics with descriptions
- **Get Pair Metrics** — Get metrics for a trading pair (e.g. btc-usd)
- **Get Exchange Metrics** — Get metrics for a specific exchange (coinbase, binance, etc.)

## API Reference

Base URL: `https://community-api.coinmetrics.io/v4`

Documentation: [CoinMetrics Community API](https://docs.coinmetrics.io/api/v4)
