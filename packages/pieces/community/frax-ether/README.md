# Frax Ether

Frax Ether is a liquid ETH staking derivative by Frax Finance. This piece provides actions to fetch protocol TVL, frxETH price, chain breakdowns, historical TVL data, and combined protocol statistics.

## Actions

- **Get Protocol TVL** - Fetch current TVL from DeFiLlama with 1h/1d/7d change percentages
- **Get frxETH Price** - Fetch frxETH price, market cap, and 24h change from CoinGecko
- **Get Chain Breakdown** - Get TVL by chain sorted descending with % of total
- **Get TVL History** - Historical TVL with configurable days parameter and % change from baseline
- **Get Protocol Stats** - Combined stats fetching TVL and frxETH price in parallel
