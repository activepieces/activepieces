# ShopSavvy Piece for Activepieces

Search products, compare prices across retailers, and track price history with ShopSavvy.

## Authentication

This piece requires a ShopSavvy API key. Get one at [shopsavvy.com/data](https://shopsavvy.com/data).

## Actions

- **Search Products** — Search for products by keyword across millions of products.
- **Get Product Details** — Look up detailed product information by barcode, ASIN, URL, model number, or product name.
- **Get Current Offers** — Get current prices and offers for a product across retailers.
- **Get Price History** — Get historical price data for a product over a date range.
- **Custom API Call** — Make a custom API call to the ShopSavvy API.

## Triggers

- **Price Drop** — Triggers when the lowest price for a monitored product drops below a threshold or decreases from its previous value. Supports optional retailer filtering and target price.
