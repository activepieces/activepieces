# Glassnode

[Glassnode](https://glassnode.com) is the leading on-chain market intelligence platform, providing institutional-grade blockchain data and analytics for Bitcoin, Ethereum, and other crypto assets.

## Actions

- **Get Active Addresses** - Retrieve the number of unique on-chain active addresses
- **Get Transactions Count** - Get the total daily on-chain transaction count
- **Get Mean Transaction Fees** - Retrieve average transaction fee data
- **Get Exchange Net Position Change** - Track supply flowing in/out of exchanges
- **Get SOPR** - Spent Output Profit Ratio (market profit/loss indicator)
