# Stride

Stride is a Cosmos liquid staking protocol. This piece provides actions to fetch TVL, token price, chain breakdown, TVL history, and protocol stats from DeFiLlama and CoinGecko.

## Actions

- **Get Protocol TVL** - Fetch the current TVL for Stride from DeFiLlama
- **Get Token Price** - Fetch the current STRD token price from CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by chain from DeFiLlama
- **Get TVL History** - Fetch the last 30 days of TVL history from DeFiLlama
- **Get Protocol Stats** - Fetch key protocol statistics from DeFiLlama

## No API Key Required

All data is fetched from public APIs (DeFiLlama and CoinGecko) with no authentication required.
