# Invariant

Invariant is a concentrated liquidity AMM DEX built on Solana that prioritizes capital efficiency, permissionless access, and composability. Similar to Uniswap v3 but native to Solana.

## Actions

- **Get Protocol TVL** - Fetch current Total Value Locked for Invariant from DeFiLlama
- **Get Token Price** - Get INV token price from CoinGecko
- **Get Chain Breakdown** - Get TVL breakdown by chain (Solana, Eclipse) from DeFiLlama
- **Get TVL History** - Get TVL history for the last 30 days from DeFiLlama
- **Get Protocol Stats** - Get key protocol statistics from DeFiLlama
