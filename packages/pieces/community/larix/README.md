# Larix

Larix is a Solana-based lending protocol offering dynamic interest rates and a multi-asset mining model. LARIX is the governance token.

## Actions

- **Get Protocol TVL** - Fetch current Total Value Locked for Larix from DeFiLlama
- **Get LARIX Price** - Fetch current LARIX token price from CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by blockchain from DeFiLlama
- **Get TVL History** - Fetch last 30 days of historical TVL data from DeFiLlama
- **Get Protocol Stats** - Fetch key protocol statistics (TVL, chains, category) from DeFiLlama

## APIs Used

- [DeFiLlama](https://defillama.com/) - Free, no API key required
- [CoinGecko](https://www.coingecko.com/) - Free public API, no key required
