# Benqi Piece for Activepieces

Integrate with [Benqi](https://benqi.fi) — a leading liquidity market protocol and liquid staking platform on Avalanche.

## Actions

- **Get Protocol TVL** — Fetch Benqi's total value locked via DeFiLlama
- **Get QI Token Price** — Fetch current QI governance token price via CoinGecko
- **Get sAVAX Price** — Fetch current sAVAX (liquid staked AVAX) price via CoinGecko
- **Get Chain Breakdown** — Fetch TVL breakdown by chain via DeFiLlama
- **Get TVL History** — Fetch last 30 days of historical TVL data via DeFiLlama

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) — Free, no auth required
- [CoinGecko API](https://www.coingecko.com/en/api) — Free tier, no auth required
