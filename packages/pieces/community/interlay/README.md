# Interlay

Interlay is a trustless Bitcoin bridge and DeFi hub on Polkadot, enabling BTC to be used in DeFi as iBTC (1:1 backed Bitcoin). INTR is the governance token.

## Actions

- **Get Protocol TVL** - Fetch current Total Value Locked for Interlay from DeFiLlama
- **Get INTR Price** - Get INTR governance token price, market cap, and volume from CoinGecko
- **Get Chain Breakdown** - Get TVL breakdown by blockchain from DeFiLlama
- **Get TVL History** - Get historical TVL data (last 30 days) from DeFiLlama
- **Get Protocol Stats** - Get key Interlay protocol statistics from DeFiLlama

## Authentication

No authentication required. Uses free public APIs:
- [DeFiLlama](https://defillama.com) - TVL data
- [CoinGecko](https://coingecko.com) - Price data
