# Mantle Network

Mantle is an EVM-compatible Ethereum Layer-2 network using optimistic rollup technology with data availability powered by EigenDA. MNT is the native governance and gas token.

## Actions

- **Get Protocol TVL**: Fetch current TVL data for Mantle from DeFiLlama
- **Get MNT Price**: Get current MNT token price and market data from CoinGecko
- **Get Chain Breakdown**: Get TVL broken down by chain from DeFiLlama
- **Get TVL History**: Get last 30 days of historical TVL data from DeFiLlama
- **Get Protocol Stats**: Get key protocol statistics (TVL, chains, category) from DeFiLlama

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) — Free, no authentication required
- [CoinGecko API](https://www.coingecko.com/en/api/documentation) — Free public tier, no authentication required
