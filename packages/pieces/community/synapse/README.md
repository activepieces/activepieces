# Synapse Protocol

ActivePieces community piece for [Synapse Protocol](https://synapseprotocol.com) — a cross-chain bridge and interoperability protocol supporting 20+ blockchains (Ethereum, Avalanche, BNB Chain, Polygon, and more). SYN is the governance token.

## Actions

- **Get Protocol TVL** — Fetch current total value locked via DeFiLlama
- **Get SYN Price** — Fetch current SYN token price via CoinGecko
- **Get Chain Breakdown** — Fetch TVL breakdown by individual chain via DeFiLlama
- **Get TVL History** — Fetch last 30 days of historical TVL data via DeFiLlama
- **Get Protocol Stats** — Fetch key protocol stats (TVL, chains, category) via DeFiLlama

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) — Free, no API key required
- [CoinGecko API](https://www.coingecko.com/en/api) — Free public tier, no API key required
