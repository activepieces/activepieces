# CoW Protocol

CoW Protocol (Coincidence of Wants) is a DEX aggregator and intent-based trading protocol that uses batch auctions to find the best prices for token swaps on Ethereum and other EVM chains.

## Actions

- **Get Protocol TVL** - Retrieve current Total Value Locked (TVL) for CoW Protocol from DeFiLlama
- **Get COW Token Price** - Get current COW governance token price and market data from CoinGecko
- **Get Chain Breakdown** - Get TVL breakdown across all supported blockchain networks
- **Get TVL History** - Retrieve historical TVL data for the last 30 days
- **Get Protocol Stats** - Get key protocol statistics including TVL, chains, and category

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) - Free, no authentication required
- [CoinGecko API](https://www.coingecko.com/en/api) - Free tier, no authentication required
