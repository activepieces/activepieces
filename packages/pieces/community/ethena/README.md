# Ethena Piece for Activepieces

This piece integrates **Ethena** protocol data into your Activepieces workflows.

Ethena is a synthetic dollar protocol built on Ethereum, offering **USDe** — a crypto-native stablecoin backed by delta-hedged ETH derivatives. **sUSDe** earns staking rewards. **ENA** is the governance token.

## Data Sources

- [DeFiLlama API](https://defillama.com/docs/api) — No API key required
- [CoinGecko API](https://www.coingecko.com/en/api) — No API key required

## Actions

| Action | Description |
|--------|-------------|
| **Get Protocol TVL** | Fetch current TVL with 1h/1d/7d percentage changes |
| **Get ENA Token Price** | Live ENA price, market cap, and 24h trading volume |
| **Get Chain TVL Breakdown** | TVL distribution across all chains |
| **Get TVL History** | Configurable historical TVL data points |
| **Get Protocol Stats** | Combined protocol and token metrics summary |
