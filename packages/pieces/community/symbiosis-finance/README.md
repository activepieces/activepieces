# Symbiosis Finance

Symbiosis Finance is a cross-chain AMM DEX that unites liquidity across any blockchain.

## Actions

| Action | Description |
|--------|-------------|
| get-protocol-tvl | Get Symbiosis Finance total value locked from DeFiLlama |
| get-token-price | Get SIS token price and market data from CoinGecko |
| get-chain-breakdown | Get TVL breakdown by chain from DeFiLlama |
| get-tvl-history | Get historical TVL data for the last 30 days from DeFiLlama |
| get-protocol-stats | Get key protocol statistics from DeFiLlama |

## Data Sources

- **DeFiLlama API** (https://api.llama.fi) — Free public API, no auth required
- **CoinGecko API** (https://api.coingecko.com/api/v3) — Free public API, no auth required

## Authentication

No API key required.
