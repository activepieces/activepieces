# Instadapp Piece for Activepieces

Integrate [Instadapp](https://instadapp.io) — the leading DeFi aggregator and smart account infrastructure — into your Activepieces workflows.

Instadapp's **DeFi Smart Accounts (DSA)** allow users to interact with multiple DeFi protocols from a single account. Their newer **Fluid** protocol is a high-efficiency lending and DEX platform.

## Actions

| Action | Description |
|---|---|
| **Get Protocol TVL** | Fetch Instadapp's total value locked (TVL) from DeFiLlama |
| **Get Fluid TVL** | Fetch Fluid protocol's total value locked from DeFiLlama |
| **Get INST Price** | Get the INST governance token price, market cap, and 24h volume from CoinGecko |
| **Get Fluid Pools** | List all Fluid lending pools with APY/yield data from DeFiLlama Yields |
| **Get Chain TVL** | Get Instadapp TVL broken down by blockchain (Ethereum, Polygon, Arbitrum, etc.) |

## Authentication

No authentication required. All data is sourced from public APIs (DeFiLlama, CoinGecko).

## Data Sources

- [DeFiLlama](https://defillama.com) — TVL and protocol data
- [CoinGecko](https://coingecko.com) — Token price and market data
- [DeFiLlama Yields](https://defillama.com/yields) — Pool yield data
