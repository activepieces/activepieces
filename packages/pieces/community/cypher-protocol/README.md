# Cypher Protocol — Activepieces Community Piece

Cypher is a **Solana-based derivatives and margin trading protocol** offering cross-margined perpetual futures and options. CYPH is the governance token that enables community governance over protocol parameters.

## Actions

| Action | Description | API |
|--------|-------------|-----|
| **Get Protocol TVL** | Fetch current Total Value Locked for Cypher Protocol | DeFiLlama |
| **Get CYPH Token Price** | Fetch live CYPH token price and market data | CoinGecko |
| **Get Chain TVL Breakdown** | Fetch TVL breakdown by blockchain | DeFiLlama |
| **Get TVL History (Last 30 Days)** | Fetch 30-day historical TVL data | DeFiLlama |
| **Get Protocol Stats** | Fetch key stats: category, chains, all-time peak TVL | DeFiLlama |

## Authentication

No API key required. All data is fetched from free public APIs:
- [DeFiLlama API](https://defillama.com/docs/api) — `https://api.llama.fi/protocol/cypher`
- [CoinGecko API](https://www.coingecko.com/en/api) — `https://api.coingecko.com/api/v3/coins/cypher-protocol`
