# Beefy Finance

Beefy Finance is a multichain yield optimizer that auto-compounds yield farming rewards across 25+ blockchains. It offers "vaults" that automatically harvest and reinvest DeFi rewards to maximize APY for users.

## Actions

- **Get Protocol TVL** - Fetch current total value locked (TVL) for Beefy Finance via DeFiLlama
- **Get BIFI Price** - Fetch the current price and market data for the BIFI governance token via CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by blockchain for Beefy Finance
- **Get TVL History** - Fetch the last 30 days of historical TVL data for Beefy Finance
- **Get Protocol Stats** - Fetch key protocol statistics including TVL, chain count, and category

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) — Free, no authentication required
- [CoinGecko API](https://www.coingecko.com/en/api) — Free tier, no authentication required
