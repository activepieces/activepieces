# X2Y2 Piece

An Activepieces community piece for [X2Y2](https://x2y2.io) — an Ethereum NFT marketplace offering zero-fee trading, staking rewards, and collection offers.

## Actions

- **Get Protocol TVL** — Fetch current total value locked for X2Y2 via DeFiLlama
- **Get X2Y2 Token Price** — Fetch X2Y2 token price and market data via CoinGecko
- **Get Chain Breakdown** — Fetch TVL breakdown by blockchain via DeFiLlama
- **Get TVL History** — Fetch last 30 days of historical TVL via DeFiLlama
- **Get Protocol Stats** — Fetch key protocol stats (TVL, chains, category) via DeFiLlama

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) — Free, no API key required
- [CoinGecko API](https://www.coingecko.com/en/api) — Free tier, no API key required
