# Filecoin Piece

Fetch data from the Filecoin decentralized storage network using DeFiLlama and CoinGecko APIs.

## Actions

| Action | Description |
|--------|-------------|
| Get Protocol TVL | Filecoin TVL with 1h/1d/7d changes |
| Get FIL Token Price | Live FIL price, market cap, 24h volume |
| Get Chain TVL Breakdown | TVL distribution across chains |
| Get TVL History | Historical TVL data points |
| Get Protocol Stats | Combined protocol + token summary |

## Data Sources
- [DeFiLlama](https://defillama.com/chain/Filecoin)
- [CoinGecko](https://www.coingecko.com/en/coins/filecoin)

No authentication required.
