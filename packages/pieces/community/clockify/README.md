# Clockify Piece for Activepieces

Time tracking and productivity platform integration.

## Features

- **Create Time Entries**: Log time for tasks automatically
- **Start/Stop Timers**: Control running timers
- **Find Tasks & Projects**: Search and retrieve data
- **Triggers**: React to new tasks and time entries

## Authentication

1. Go to [Clockify](https://clockify.me)
2. Navigate to Profile Settings
3. Generate API Key
4. Copy and paste into Activepieces

## Actions

### Create Time Entry
Log time for a specific project and task.

### Start Timer
Begin a new timer for time tracking.

### Stop Timer
Stop currently running timer.

### Find Task
Search for a specific task by name or ID.

### Find Time Entry
Retrieve time entries with filters.

## Triggers

### New Task
Triggers when a new task is created in a project.

### New Time Entry
Fires when a new time entry is logged.

### New Timer Started
Activates when a timer begins.
