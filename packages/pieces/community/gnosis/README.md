# Gnosis Chain (xDAI) Piece for Activepieces

Integrate [Gnosis Chain](https://www.gnosis.io/) — an EVM-compatible L1 blockchain using **xDAI** (bridged DAI) as the native gas token — into your Activepieces workflows.

## About Gnosis Chain

Gnosis Chain (formerly xDai Chain) is designed for fast, inexpensive, and stable transactions. It uses xDAI (pegged to $1) as gas, making it ideal for everyday transactions and decentralized applications. GNO is the governance and staking token.

## Actions

| Action | Description | API Source |
|--------|-------------|------------|
| **Get Protocol TVL** | Total value locked on Gnosis Chain from DeFiLlama | DeFiLlama |
| **Get GNO Price** | Current GNO token price, market cap, and 24h volume | CoinGecko |
| **Get Chain Breakdown** | TVL breakdown by chain for all protocols on Gnosis | DeFiLlama |
| **Get TVL History** | Last 30 days of historical TVL data for Gnosis Chain | DeFiLlama |
| **Get Protocol Stats** | Key protocol stats: TVL, chains supported, category | DeFiLlama |

## Authentication

No API key required — all data is sourced from free public APIs (DeFiLlama & CoinGecko).
