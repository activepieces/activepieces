# Umee

Activepieces community piece for [Umee](https://umee.cc) — a cross-chain DeFi lending protocol.

## Actions

- **Get Protocol TVL** — Fetch current TVL from DeFiLlama
- **Get Token Price** — Fetch UMEE token price from CoinGecko
- **Get Chain TVL Breakdown** — TVL by chain from DeFiLlama
- **Get TVL History (Last 30 Days)** — Historical TVL data from DeFiLlama
- **Get Protocol Stats** — Key protocol statistics from DeFiLlama

## No API Key Required
