# GoodMem Piece

This piece provides integration with GoodMem, a vector-based memory storage and semantic retrieval system for AI applications.

## Actions

### Create Memory
Create a new memory in a specified space with content and metadata.

### Retrieve Memories
Perform semantic search to retrieve relevant memories based on a natural language query.

### Get Memory
Retrieve a specific memory by its ID, with optional content inclusion.

### Delete Memory
Delete a specific memory by its ID.

## Authentication

This piece uses Custom Authentication with the following properties:

- **Base URL**: The base URL of your GoodMem instance (e.g., `http://localhost:8080`)
- **API Key**: Your GoodMem API key

## Requirements

You need a running GoodMem instance with an API key to use this piece.
