# Meteora

Meteora is a dynamic liquidity protocol on Solana featuring Dynamic AMM pools, DLMM (Dynamic Liquidity Market Maker) with concentrated liquidity bins, and yield-bearing vaults. MET is the governance token.

## Actions

- **Get Protocol TVL** - Fetch the current Total Value Locked (TVL) for Meteora from DeFiLlama
- **Get MET Price** - Fetch the current price and market data for the MET governance token from CoinGecko
- **Get Chain Breakdown** - Fetch Meteora's TVL breakdown by blockchain chain
- **Get TVL History** - Fetch the last 30 days of historical TVL data for Meteora
- **Get Protocol Stats** - Fetch key protocol statistics including TVL, chains, and category
