# NEAR Protocol Piece

This piece integrates with NEAR Protocol, a Layer-1 blockchain featuring Nightshade sharding for scalability, human-readable account names, and low transaction fees.

## Actions

- **Get Protocol TVL** - Fetch current Total Value Locked for NEAR Protocol from DeFiLlama
- **Get NEAR Price** - Fetch current NEAR token price and market data from CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by chain from DeFiLlama
- **Get TVL History** - Fetch last 30 days of historical TVL data from DeFiLlama
- **Get Protocol Stats** - Fetch key NEAR Protocol statistics (TVL, chains, category) from DeFiLlama

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) - Free, no authentication required
- [CoinGecko API](https://www.coingecko.com/en/api) - Free tier, no authentication required
