# Wealthbox CRM

Wealthbox is a modern CRM platform designed specifically for financial advisors.

## Authentication

This piece uses API Key authentication. To get your API key:

1. Log in to your Wealthbox account
2. Navigate to Settings > API & Integrations
3. Generate a new API access token
4. Copy the API token and use it in Activepieces

## Triggers

- **New Task** - Triggers when a new task is created
- **New Contact** - Triggers when a new contact is created
- **New Event** - Triggers when a new event is created
- **New Opportunity** - Triggers when a new opportunity is created

## Actions

- **Create Contact** - Create a new contact with name, email, tags, and more
- **Create Note** - Add a note linked to a contact
- **Create Project** - Start a new project
- **Create Household** - Create a household record
- **Add Member to Household** - Add a member to an existing household
- **Create Event** - Create a calendar event
- **Create Opportunity** - Log an opportunity with stage and amount
- **Create Task** - Create a task with due date
- **Start Workflow** - Trigger a workflow on a contact or project
- **Find Contact** - Search for contacts
- **Find Task** - Search for tasks
