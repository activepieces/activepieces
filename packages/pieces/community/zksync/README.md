# zkSync Era Piece

Activepieces community piece for zkSync Era — a ZK rollup Layer-2 on Ethereum by Matter Labs.

## Actions

- **Get Protocol TVL** — Fetch current Total Value Locked for zkSync Era from DeFiLlama
- **Get ETH Price** — Fetch current ETH price from CoinGecko (ETH is the gas token on zkSync Era)
- **Get Chain Breakdown** — Get TVL breakdown by chain for zkSync Era from DeFiLlama
- **Get TVL History** — Fetch last 30 days of historical TVL data from DeFiLlama
- **Get Protocol Stats** — Get key protocol statistics (TVL, chains, category) from DeFiLlama

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) — Free, no API key required
- [CoinGecko API](https://www.coingecko.com/en/api) — Free, no API key required
