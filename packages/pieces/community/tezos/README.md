# Tezos Piece for Activepieces

Interact with the Tezos blockchain ecosystem using free public APIs (no API key required).

## Actions

- **Get Protocol TVL** - Fetch total value locked for the Tezos protocol via DeFiLlama
- **Get XTZ Price** - Fetch current XTZ price, market cap and volume via CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by chain/protocol for Tezos via DeFiLlama
- **Get TVL History** - Fetch last 30 days of historical TVL data for Tezos via DeFiLlama
- **Get Protocol Stats** - Fetch key protocol stats (TVL, chains, category) via DeFiLlama

## APIs Used

- DeFiLlama API - Free, no authentication required
- CoinGecko API - Free tier, no authentication required
