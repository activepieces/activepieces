# Francium

Francium is a leveraged yield farming protocol on Solana, allowing users to earn amplified yields by borrowing additional capital. FRW is the governance token.

## Actions

- **Get Protocol TVL** - Fetches the current Total Value Locked (TVL) for the Francium protocol from DeFiLlama.
- **Get FRW Token Price** - Fetches the current FRW token price and market data from CoinGecko.
- **Get Chain TVL Breakdown** - Fetches the TVL breakdown by blockchain for the Francium protocol from DeFiLlama.
- **Get TVL History (Last 30 Days)** - Fetches the historical TVL data for Francium over the last 30 days from DeFiLlama.
- **Get Protocol Stats** - Fetches key Francium protocol statistics including TVL, chains, category, and more from DeFiLlama.

## APIs Used

- [DeFiLlama](https://defillama.com/docs/api) - Free, no authentication required
- [CoinGecko](https://www.coingecko.com/en/api/documentation) - Free public API, no authentication required
