# Helium Piece

Fetch data from the Helium Network using DeFiLlama and CoinGecko APIs.

## Actions

| Action | Description |
|--------|-------------|
| Get Protocol TVL | Helium TVL with 1h/1d/7d changes |
| Get HNT Token Price | Live HNT price, market cap, 24h volume |
| Get Chain TVL Breakdown | TVL distribution across chains |
| Get TVL History | Historical TVL data points |
| Get Protocol Stats | Combined protocol + token summary |

## Data Sources
- [DeFiLlama](https://defillama.com/protocol/helium)
- [CoinGecko](https://www.coingecko.com/en/coins/helium)

No authentication required.
