# Lybra Finance

Lybra Finance is a leading LSDfi (Liquid Staking DeFi) protocol that enables users to stake stETH/eETH as collateral to mint eUSD — a yield-bearing stablecoin that automatically appreciates in value. LBR is the governance and utility token.

## Actions

- **Get Protocol TVL** — Fetch total value locked from DeFiLlama
- **Get LBR Price** — Fetch live LBR token price from CoinGecko
- **Get eUSD Price** — Fetch live eUSD stablecoin price from CoinGecko
- **Get Chain Breakdown** — Get TVL breakdown by chain from DeFiLlama
- **Get TVL History** — Fetch last 30 days of historical TVL from DeFiLlama

No API key required. All data is publicly available.
