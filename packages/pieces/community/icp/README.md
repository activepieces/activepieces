# ICP (Internet Computer Protocol)

Actions for the Internet Computer Protocol (ICP) — a Layer-1 blockchain by DFINITY that runs smart contracts ("canisters") at web speed with unlimited capacity.

## Actions

- **Get Protocol TVL** — Fetch current Total Value Locked for ICP from DeFiLlama
- **Get ICP Price** — Fetch current ICP token price and market data from CoinGecko
- **Get Chain Breakdown** — Fetch TVL breakdown by chain from DeFiLlama
- **Get TVL History** — Fetch last 30 days of historical TVL data from DeFiLlama
- **Get Protocol Stats** — Fetch key protocol statistics (TVL, chains, category) from DeFiLlama

All actions use free public APIs with no authentication required.
