# @activepieces/piece-centrifuge

Activepieces community piece for [Centrifuge](https://centrifuge.io) - the real-world asset (RWA) tokenization protocol.

## Actions

- **Get Protocol TVL** - Current TVL from DeFiLlama
- **Get CFG Price** - CFG token price from CoinGecko
- **Get Chain Breakdown** - TVL per chain
- **Get TVL History** - Historical TVL (configurable days)
- **Get Protocol Stats** - Combined TVL + price snapshot

## APIs Used

- [DeFiLlama](https://defillama.com) - public, no auth required
- [CoinGecko](https://coingecko.com) - public, no auth required
