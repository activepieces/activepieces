# Hedera HBAR Piece

Activepieces community piece for Hedera — the enterprise-grade public distributed ledger using hashgraph consensus.

## Actions

- **Get Protocol TVL** — Fetch total value locked on Hedera via DeFiLlama
- **Get HBAR Price** — Current USD price, market cap, and 24h volume of HBAR via CoinGecko
- **Get Chain Breakdown** — TVL breakdown by chain via DeFiLlama
- **Get TVL History** — Last 30 days of historical TVL via DeFiLlama
- **Get Protocol Stats** — Key stats (TVL, chains, category) via DeFiLlama

## APIs

- [DeFiLlama](https://defillama.com/) — Free, no auth required
- [CoinGecko](https://www.coingecko.com/) — Free, no auth required
