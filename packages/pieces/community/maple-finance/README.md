# Maple Finance

Maple Finance is an institutional capital marketplace on Ethereum and Solana, enabling undercollateralized lending to institutions through lending pools managed by Pool Delegates.

## Actions

- **Get Protocol TVL** - Fetch current Total Value Locked across all Maple pools
- **Get MPL Price** - Get MPL governance token price and market data
- **Get SYRUP Price** - Get SYRUP token price and market data
- **Get Chain Breakdown** - TVL breakdown by blockchain
- **Get TVL History** - Historical TVL for the last 30 days

No API key required.
