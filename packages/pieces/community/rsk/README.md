# Rootstock (RSK) Piece

Activepieces integration for Rootstock (RSK) — the Bitcoin sidechain enabling EVM-compatible smart contracts secured by Bitcoin's hashrate.

## Actions

- **Get Protocol TVL** — Fetch total value locked on Rootstock via DeFiLlama
- **Get RBTC Price** — Get the current RBTC price (pegged to BTC) via CoinGecko
- **Get Chain Breakdown** — Fetch TVL breakdown by chain from DeFiLlama
- **Get TVL History** — Retrieve last 30 days of historical TVL data
- **Get Protocol Stats** — Get key Rootstock protocol statistics (TVL, chains, category)

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) — Free, no authentication required
- [CoinGecko API](https://www.coingecko.com/en/api) — Free tier, no authentication required
