# Blur NFT Marketplace

This piece integrates with the Blur NFT marketplace protocol via DeFiLlama and CoinGecko free public APIs.

## Actions

- **Get Protocol TVL**: Fetch current TVL for the Blur protocol from DeFiLlama
- **Get BLUR Price**: Fetch current BLUR token price from CoinGecko
- **Get Chain Breakdown**: Fetch TVL breakdown by blockchain
- **Get TVL History**: Fetch last 30 days of historical TVL data
- **Get Protocol Stats**: Fetch key protocol stats (TVL, chains, category)

No API key required.
