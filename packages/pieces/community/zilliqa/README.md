# Zilliqa Piece for Activepieces

This piece integrates with the Zilliqa blockchain ecosystem using free public APIs (DeFiLlama and CoinGecko). No API key required.

## Actions

- **Get Protocol TVL** - Fetch total value locked for Zilliqa from DeFiLlama
- **Get ZIL Price** - Fetch current ZIL token price from CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by chain for Zilliqa
- **Get TVL History** - Fetch last 30 days of historical TVL data
- **Get Protocol Stats** - Fetch key stats: TVL, chains, category info
