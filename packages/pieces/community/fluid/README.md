# Fluid (Instadapp) Piece for Activepieces

**Fluid** is a next-generation DeFi protocol by [Instadapp](https://instadapp.io) featuring an innovative AMM DEX and lending protocol with smart collateral/debt features. INST is the governance token.

## Actions

| Action | Description | API |
|--------|-------------|-----|
| Get Protocol TVL | Total value locked across all chains | DeFiLlama |
| Get INST Token Price | Price, market cap, 24h volume, changes | CoinGecko |
| Get TVL by Chain | TVL breakdown by blockchain | DeFiLlama |
| Get TVL History (30 Days) | Historical TVL for last 30 days | DeFiLlama |
| Get Protocol Stats | Key stats: TVL, chains, category, 24h/7d change | DeFiLlama |

## Authentication

No authentication required. All data is sourced from public DeFiLlama and CoinGecko APIs.

## Use Cases

- Monitor Fluid protocol health in automated workflows
- Track INST token price for portfolio alerts
- Compare TVL across chains for DeFi research
- Historical TVL analysis and trend tracking
- Dashboard reporting on DeFi protocol statistics
