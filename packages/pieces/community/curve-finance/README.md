# Curve Finance

Curve Finance is the leading decentralized exchange (DEX) and AMM protocol specialized in stablecoin and pegged-asset swaps. It uses a unique bonding curve optimized for assets that trade at similar values, offering extremely low slippage for stablecoin trades. CRV is the governance token with vote-escrowed veCRV mechanics for yield boosting and protocol governance.

## Actions

- **Get Protocol TVL** — Current TVL from DeFiLlama with 1h/1d/7d changes
- **Get CRV Token Price** — Live CRV price, market cap, 24h volume from CoinGecko
