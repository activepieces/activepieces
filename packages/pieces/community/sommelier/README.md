# Sommelier Finance

Sommelier Finance is a DeFi protocol that offers actively managed vaults using off-chain strategy computation combined with on-chain execution. SOMM is the governance token.

## Actions

- **Get Protocol TVL** - Fetch the current Total Value Locked (TVL) for Sommelier Finance from DeFiLlama
- **Get SOMM Price** - Fetch the current price and market data for SOMM token from CoinGecko
- **Get Chain Breakdown** - Get Sommelier's TVL broken down by blockchain from DeFiLlama
- **Get TVL History** - Fetch the last 30 days of historical TVL data for Sommelier from DeFiLlama
- **Get Protocol Stats** - Get key protocol statistics including TVL, chains count, and category from DeFiLlama

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) - Free, no authentication required
- [CoinGecko API](https://www.coingecko.com/en/api) - Free tier, no authentication required
