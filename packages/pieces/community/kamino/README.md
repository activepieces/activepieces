# Kamino Finance

Kamino Finance is a leading DeFi protocol on Solana offering automated concentrated liquidity strategies, lending markets, and leveraged yield. KMNO is the governance token.

## Actions

- **Get Protocol TVL** - Fetch current Total Value Locked for Kamino Finance via DeFiLlama
- **Get KMNO Price** - Fetch the current price and market data for the KMNO governance token via CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by blockchain chain for Kamino Finance
- **Get TVL History** - Fetch historical TVL data for Kamino Finance over the last 30 days
- **Get Protocol Stats** - Fetch key protocol statistics including TVL, chains, and category

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) - Free, no authentication required
- [CoinGecko API](https://www.coingecko.com/en/api) - Free tier, no authentication required
