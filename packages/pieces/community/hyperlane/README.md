# Hyperlane

Hyperlane is a permissionless interoperability protocol that allows anyone to connect any blockchain. It enables cross-chain communication and asset transfers without requiring permission from any central authority. HYPER is the governance token of the Hyperlane protocol.

## Actions

- **Get Protocol TVL** - Fetch total value locked in Hyperlane protocol via DeFiLlama
- **Get HYPER Price** - Get current HYPER token price and market data via CoinGecko
- **Get Chain Breakdown** - Get TVL breakdown by chain for Hyperlane protocol
- **Get TVL History** - Fetch last 30 days of historical TVL data for Hyperlane
- **Get Protocol Stats** - Get key protocol statistics including TVL, chains count, and category

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) - Free, no authentication required
- [CoinGecko API](https://www.coingecko.com/en/api/documentation) - Free tier, no authentication required
