# Aurora NEAR EVM Piece

Aurora is an EVM-compatible Layer-2 built on NEAR Protocol. This piece provides actions to fetch Aurora protocol data from DeFiLlama and CoinGecko.

## Actions

- **Get Protocol TVL** - Fetch current TVL data for Aurora from DeFiLlama
- **Get Aurora Price** - Get current AURORA token price and market data from CoinGecko
- **Get Chain Breakdown** - Get TVL breakdown by chain for Aurora from DeFiLlama
- **Get TVL History** - Fetch last 30 days of historical TVL data for Aurora
- **Get Protocol Stats** - Get key protocol statistics (TVL, chains, category) for Aurora

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) - Free, no API key required
- [CoinGecko API](https://www.coingecko.com/en/api) - Free tier, no API key required
