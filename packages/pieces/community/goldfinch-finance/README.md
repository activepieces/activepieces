# Goldfinch Finance

Activepieces community piece for [Goldfinch Finance](https://goldfinch.finance) — a decentralized credit protocol for real-world asset (RWA) lending.

## Actions

- **Get Protocol TVL** — Fetch total value locked from DeFiLlama
- **Get GFI Price** — Fetch GFI token price from CoinGecko
- **Get Chain Breakdown** — Fetch per-chain TVL breakdown from DeFiLlama
- **Get TVL History** — Fetch historical TVL data from DeFiLlama
- **Get Protocol Stats** — Combined protocol stats (TVL + GFI price)

No authentication required. Uses public DeFiLlama and CoinGecko APIs.
