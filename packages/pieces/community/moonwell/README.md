# Moonwell Piece for Activepieces

Moonwell is a leading open lending and borrowing protocol on Base (Coinbase L2) and Moonbeam, backed by Coinbase Ventures. WELL is the governance token.

## Actions

- **Get Protocol TVL** — Fetches the current Total Value Locked (TVL) for Moonwell from DeFiLlama, including 1h, 1d, and 7d percentage changes.
- **Get WELL Token Price** — Retrieves live WELL token price, market cap, 24h volume, and price change data from CoinGecko.
- **Get Chain TVL Breakdown** — Returns TVL breakdown by chain (Base, Moonbeam, etc.) from DeFiLlama.
- **Get TVL History** — Returns historical TVL data points for Moonwell, with a configurable limit.
- **Get Protocol Stats** — Combined summary of Moonwell protocol metrics and WELL token data from both DeFiLlama and CoinGecko.

## Data Sources

- [DeFiLlama API](https://defillama.com/docs/api) (no authentication required)
- [CoinGecko API](https://www.coingecko.com/en/api) (no authentication required)

## About Moonwell

Moonwell is an open and permissionless lending and borrowing DeFi protocol built on Base and Moonbeam. It allows users to supply assets to earn yield and borrow against their collateral. Backed by Coinbase Ventures, Moonwell has grown to become one of the leading protocols on Base.

- Website: [moonwell.fi](https://moonwell.fi)
- Token: WELL (governance token)
- Networks: Base (Coinbase L2), Moonbeam
