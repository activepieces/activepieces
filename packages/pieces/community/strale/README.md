# Strale

Trust layer for AI agents. 271 quality-scored API capabilities for company verification, compliance checks, financial validation, web intelligence, and more.

## Actions

- **Search Capabilities** — Find the right capability by keyword or category. No API key required.
- **Execute Capability** — Run any capability by slug. Returns structured output with quality score and data provenance.
- **Check Balance** — View wallet balance before executing paid capabilities.
- **Trust Profile** — Check a capability's quality score, reliability grade, and execution guidance before relying on it.
- **Custom API Call** — Make arbitrary requests to the Strale API.

## Authentication

API key from [strale.dev/signup](https://strale.dev/signup). Free €2.00 trial credits, no card required.

Search and Trust Profile work without an API key. Five free capabilities (email-validate, dns-lookup, json-repair, url-to-markdown, iban-validate) work without signup.
