# Velodrome Finance — Activepieces Piece

Velodrome Finance is the leading DEX on Optimism, inspired by Curve and Convex. It uses a ve(3,3) tokenomics model and has the deepest liquidity on Optimism.

## Actions

| Action | Description |
|--------|-------------|
| **Get Protocol TVL** | Get Velodrome total value locked on Optimism (DeFiLlama) |
| **Get VELO Price** | Get VELO token price, market cap, and 24h volume (CoinGecko) |
| **Get Top Pools** | Get top liquidity pools sorted by TVL (DeFiLlama yields) |
| **Get Pool APYs** | Get pool APY data with optional minimum APY filter (DeFiLlama yields) |
| **Get Chain Stats** | Get Optimism chain statistics from DeFiLlama |

## Authentication

No authentication required — all data sourced from free public APIs:
- [DeFiLlama](https://defillama.com) (`api.llama.fi`, `yields.llama.fi`)
- [CoinGecko](https://coingecko.com) (public endpoint)
