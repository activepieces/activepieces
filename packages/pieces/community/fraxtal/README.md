# Fraxtal

Fraxtal is a modular blockchain and EVM-equivalent rollup built by Frax Finance. It uses FRAX tokens as gas and features FXTL points for incentivized usage.

## Actions

- **Get Protocol TVL** - Fetch total value locked for the Fraxtal protocol from DeFiLlama
- **Get FRAX Price** - Get current FRAX token price data from CoinGecko
- **Get Chain Breakdown** - Get TVL breakdown by chain from DeFiLlama
- **Get TVL History** - Fetch last 30 days of historical TVL data from DeFiLlama
- **Get Protocol Stats** - Get key protocol statistics (TVL, chains, category) from DeFiLlama

All actions use free public APIs with no authentication required.
