# Ampeco ActivePieces Integration

This piece provides integration with the Ampeco API for managing EV charging infrastructure.

## API Version

This integration is built against Ampeco API version: **3.96.4**
API: AMPECO.CHARGE Public API

## Features

- 376 actions for interacting with the Ampeco API
- 1 triggers for responding to Ampeco events

## Authentication

This piece uses API token (bearer) authentication. You will need to:

1. Create API token in your Ampeco account
2. Use the API token and your tenant URL when configuring the piece in ActivePieces

## Actions

This piece includes actions for managing various aspects of your Ampeco account, including:

- Chargers and charging stations
- Users and accounts
- Payments and billing
- Reports and analytics

## Triggers

- Webhook trigger for receiving real-time events from Ampeco
