# IOTA Piece for Activepieces

This piece integrates with [IOTA](https://www.iota.org/), a distributed ledger technology using a DAG-based "Tangle" designed for IoT microtransactions.

## Actions

- **Get Protocol TVL** - Fetch current Total Value Locked for IOTA via DeFiLlama
- **Get IOTA Price** - Fetch current IOTA token price data via CoinGecko
- **Get Chain Breakdown** - Fetch TVL broken down by chain via DeFiLlama
- **Get TVL History** - Fetch last 30 days of historical TVL data via DeFiLlama
- **Get Protocol Stats** - Fetch key protocol statistics (TVL, chains, category) via DeFiLlama

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) - Free, no authentication required
- [CoinGecko API](https://www.coingecko.com/en/api) - Free tier, no authentication required
