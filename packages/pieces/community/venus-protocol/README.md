# Venus Protocol Piece

Venus Protocol is the leading decentralized money market on BNB Chain, offering lending, borrowing, and synthetic stablecoin (VAI) minting. XVS is the governance token.

## Actions

### Get Protocol TVL
Fetch the current Total Value Locked (TVL) for Venus Protocol from DeFiLlama, including 1h, 1d, and 7d changes.

### Get XVS Token Price
Fetch the current price, market cap, and 24h trading volume for the XVS governance token from CoinGecko.

### Get Chain TVL Breakdown
Fetch the TVL breakdown across all chains (BNB Chain and others) for Venus Protocol from DeFiLlama.

### Get TVL History
Fetch historical TVL data for Venus Protocol from DeFiLlama, with a configurable number of data points.

### Get Protocol Stats
Fetch a combined summary of protocol metrics (TVL, chain breakdown) and XVS token metrics (price, market cap, volume) in one action.

## Data Sources

- [DeFiLlama API](https://defillama.com/docs/api) — no authentication required
- [CoinGecko API](https://www.coingecko.com/en/api) — no authentication required

## About Venus Protocol

Venus Protocol is the #1 money market and synthetic stablecoin protocol built on BNB Chain. It enables users to:
- Supply assets as collateral
- Borrow against collateral
- Mint VAI synthetic stablecoin
- Earn XVS governance rewards
