# Li.Fi Piece for Activepieces

Li.Fi is a cross-chain bridge and DEX aggregator that finds the best routes across 20+ chains. LIFI is the governance token.

## Actions

- **Get Protocol TVL** - Fetch current Total Value Locked for Li.Fi from DeFiLlama
- **Get LIFI Token Price** - Fetch current LIFI token price and market data from CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by blockchain for Li.Fi from DeFiLlama
- **Get TVL History** - Fetch last 30 days of historical TVL data for Li.Fi from DeFiLlama
- **Get Protocol Stats** - Fetch key protocol statistics (TVL, chains, category) from DeFiLlama

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) — Free, no API key required
- [CoinGecko API](https://www.coingecko.com/en/api) — Free tier, no API key required
