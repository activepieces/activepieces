# VeChain Piece for Activepieces

This piece integrates VeChain blockchain data into your Activepieces workflows.

## Actions

- **Get Protocol TVL** - Fetch VeChain's current Total Value Locked from DeFiLlama
- **Get VET Price** - Get the current VET token price and market data from CoinGecko
- **Get VTHO Price** - Get the current VTHO (VeThor) token price and market data from CoinGecko
- **Get Chain TVL Breakdown** - Retrieve TVL breakdown by protocol on the VeChain chain
- **Get TVL History** - Fetch the last 30 days of historical TVL data for VeChain

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) - Free, no authentication required
- [CoinGecko API](https://www.coingecko.com/en/api) - Free tier, no authentication required
