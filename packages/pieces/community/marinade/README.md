# Marinade Finance

Marinade Finance is the leading liquid staking protocol on Solana. Users stake SOL to receive mSOL (marinade staked SOL) which accrues staking rewards automatically.

## Actions

- **Get Protocol TVL** - Fetch total value locked in Marinade Finance via DeFiLlama
- **Get MNDE Price** - Fetch current MNDE governance token price via CoinGecko
- **Get mSOL Price** - Fetch current mSOL liquid staking token price via CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by blockchain from DeFiLlama
- **Get TVL History** - Fetch last 30 days of historical TVL from DeFiLlama

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) - Free, no authentication required
- [CoinGecko API](https://www.coingecko.com/en/api) - Free tier, no authentication required
