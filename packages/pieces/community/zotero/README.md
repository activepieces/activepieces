# Zotero

Zotero is a free, open-source reference manager used by researchers, students, and academics worldwide to collect, organize, annotate, and share research.

## Authentication

Generate a personal API key at [zotero.org/settings/keys/new](https://www.zotero.org/settings/keys/new). Select **Read/Write** access for your library. Your **Library ID** is shown on the same settings page.

## Actions

- **Get Items** — Retrieve items from your library, optionally filtered by collection, item type, and sort order.
- **Create Item** — Add a new reference (webpage, article, book, etc.) to your Zotero library.

## Triggers

- **New Item Added** — Fires when a new item is added to your Zotero library, with full pagination support for bulk imports.
