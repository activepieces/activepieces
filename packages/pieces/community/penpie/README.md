# Penpie

Penpie is a yield booster protocol built on top of Pendle Finance, similar to how Convex is built on top of Curve. Users deposit Pendle LP tokens into Penpie to earn boosted yields. PNP is the governance token.

## Actions

- **Get Protocol TVL** - Fetch the current total value locked in Penpie via DeFiLlama
- **Get PNP Price** - Fetch current PNP token price and market data via CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by blockchain network
- **Get TVL History** - Fetch last 30 days of historical TVL data
- **Get Protocol Stats** - Fetch key protocol statistics including TVL, chains, and category

All actions use free public APIs and require no authentication.
