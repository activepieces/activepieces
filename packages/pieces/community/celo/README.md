# Celo

Celo is an EVM-compatible mobile-first blockchain focused on financial inclusion, featuring a stablecoin ecosystem (cUSD, cEUR, cREAL). CELO is the native governance token.

## Actions

- **Get Protocol TVL** - Fetch total value locked for the Celo protocol from DeFiLlama
- **Get CELO Price** - Get current CELO token price and market data from CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by chain for Celo from DeFiLlama
- **Get TVL History** - Fetch last 30 days of historical TVL data from DeFiLlama
- **Get Protocol Stats** - Fetch key protocol statistics (TVL, chains, category) from DeFiLlama
