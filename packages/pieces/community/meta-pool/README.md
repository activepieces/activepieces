# Meta Pool

Meta Pool is a liquid staking protocol built on NEAR Protocol and Ethereum.

## Features

- **stNEAR**: Liquid staking token on NEAR Protocol
- **mpETH**: Liquid staking token on Ethereum
- **mpDAO**: Governance token (formerly META)

## Actions

### get-protocol-tvl
Get the current Total Value Locked (TVL) for Meta Pool from DeFiLlama, including 1h, 1d, and 7d percentage changes.

### get-meta-price
Get the current mpDAO (META) token price, market cap, and 24h price change from CoinGecko.

### get-chain-breakdown
Get the TVL breakdown by blockchain, sorted by TVL descending, with percentage of total.

### get-tvl-history
Get historical TVL data with configurable number of days and percentage change from baseline.

### get-protocol-stats
Get a combined overview of Meta Pool protocol stats including TVL and token price in a single call.
