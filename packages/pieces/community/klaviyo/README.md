# @activepieces/piece-klaviyo

Klaviyo integration for Activepieces.

## Actions

- Create Profile
- Update Profile
- Subscribe Profile
- Unsubscribe Profile
- Add Profile to List
- Remove Profile from List
- Create List
- Find Profile by Email/Phone
- Find List by Name
- Find Tag by Name

## Triggers

- New Profile
- Profile Added to List/Segment
