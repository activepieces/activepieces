# deBridge Piece for Activepieces

[deBridge](https://debridge.finance) is a cross-chain interoperability protocol enabling secure, high-performance transfers of assets and data across blockchains.

## Actions

| Action | Description |
|--------|-------------|
| **Get Protocol TVL** | Get deBridge total value locked across all chains from DeFiLlama |
| **Get DBR Price** | Get DBR token price, market cap, and 24h volume from CoinGecko |
| **Get Bridge Stats** | Get bridge TVL stats and chain breakdown from DeFiLlama |
| **Get Chain TVL** | Get TVL for a specific blockchain from DeFiLlama |
| **Get TVL History** | Get historical TVL data for the last 30 days |

## Data Sources

- **DeFiLlama API** — Free, no auth required. Provides protocol TVL, chain breakdowns, and historical data.
- **CoinGecko API** — Free, no auth required. Provides token price, market cap, and volume data.

## Authentication

No authentication required. All APIs are publicly accessible.
