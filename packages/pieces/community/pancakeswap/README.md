# PancakeSwap Piece for Activepieces

![PancakeSwap Logo](https://cdn.activepieces.com/pieces/pancakeswap.png)

Integrate **PancakeSwap** — the leading decentralized exchange (DEX) on BNB Chain (BSC) — into your Activepieces automations.

## What is PancakeSwap?

PancakeSwap is the #1 DEX on BNB Chain by trading volume and TVL. It powers billions in liquidity across hundreds of trading pairs, with its native CAKE token driving governance and yield farming.

## Actions

| Action | Description |
|--------|-------------|
| **Get Protocol TVL** | Fetch PancakeSwap's total value locked across all chains via DeFiLlama |
| **Get CAKE Price** | Fetch CAKE token price, market cap, and 24h volume from CoinGecko |
| **Get Top Pools** | Fetch the highest-TVL PancakeSwap v3 pools from DeFiLlama Yields |
| **Get Pool APYs** | Fetch pool APY data with optional minimum APY filter |
| **Get Chain TVL** | Fetch BNB Chain (BSC) total TVL from DeFiLlama |

## Authentication

No API key required. All data is sourced from public endpoints:
- [DeFiLlama API](https://defillama.com/docs/api)
- [CoinGecko API](https://www.coingecko.com/en/api)

## Example Use Cases

- Monitor PancakeSwap TVL and alert when it drops below a threshold
- Track CAKE price and send notifications on significant moves
- Build a dashboard of top-yielding BSC liquidity pools
- Automate DeFi reports combining PancakeSwap + BNB Chain data
