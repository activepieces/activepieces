# Nutshell CRM

Activepieces piece for Nutshell CRM integration.

## Actions

- **Leads:** Create, Get, Update, Search
- **Contacts:** Create, Get, Update, Search
- **Companies:** Create, Get, Update, Search
- **Custom API Call**

## Auth

Basic Auth (email + API key). Get your API key from Nutshell Setup > API & Integrations.
