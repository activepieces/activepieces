# Astar Network

Astar Network is an EVM+WASM smart contract platform on Polkadot, supporting multi-chain dApps. ASTR is the native token.

## Actions

- **Get Protocol TVL** - Fetch total value locked for Astar via DeFiLlama
- **Get ASTR Price** - Fetch current ASTR token price via CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by chain via DeFiLlama
- **Get TVL History** - Fetch last 30 days of historical TVL via DeFiLlama
- **Get Protocol Stats** - Fetch key protocol statistics via DeFiLlama
