# Solend

Solend is the largest lending and borrowing protocol on Solana, allowing users to earn interest and borrow against their crypto. SLND is the governance token.

## Actions

- **Get Protocol TVL** - Fetch current Total Value Locked for Solend from DeFiLlama
- **Get SLND Price** - Fetch current SLND token price from CoinGecko
- **Get Chain Breakdown** - Fetch Solend's TVL breakdown by blockchain
- **Get TVL History** - Fetch Solend's historical TVL for the last 30 days
- **Get Protocol Stats** - Fetch key protocol statistics (TVL, chains, category)

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) - Free, no authentication required
- [CoinGecko API](https://www.coingecko.com/api/documentation) - Free tier, no authentication required
