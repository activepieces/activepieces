# Klaytn (Kaia) Piece for Activepieces

Klaytn (now rebranding to **Kaia**) is a high-performance EVM-compatible Layer-1 blockchain built by Kakao, popular in Korea and Asia. KLAY is the native token.

## Actions

| Action | Description | API |
|--------|-------------|-----|
| Get Protocol TVL | Current Total Value Locked for Klaytn | DeFiLlama |
| Get KLAY Price | Current KLAY price and market data | CoinGecko |
| Get Chain TVL Breakdown | TVL broken down by chain | DeFiLlama |
| Get TVL History (30 Days) | Historical TVL over the last 30 days | DeFiLlama |
| Get Protocol Stats | Key stats: TVL, chains, category, description | DeFiLlama |

## Authentication

No API key required. All endpoints are free and public.

## APIs Used

- [DeFiLlama](https://defillama.com/docs/api) — `https://api.llama.fi/protocol/klaytn`
- [CoinGecko](https://www.coingecko.com/en/api) — `https://api.coingecko.com/api/v3/coins/klay-token`
