# Sudoswap

Activepieces integration for [Sudoswap](https://sudoswap.xyz), the Ethereum NFT AMM protocol enabling automated on-chain NFT trading with custom bonding curves.

## Actions

- **Get Protocol TVL** - Fetch current Total Value Locked for the Sudoswap protocol via DeFiLlama
- **Get SUDO Price** - Fetch current SUDO governance token price and market data via CoinGecko
- **Get Chain Breakdown** - Fetch TVL breakdown by blockchain for the Sudoswap protocol
- **Get TVL History** - Fetch last 30 days of historical TVL data for Sudoswap
- **Get Protocol Stats** - Fetch key protocol statistics including TVL, chains, and category

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) - Free, no API key required
- [CoinGecko API](https://www.coingecko.com/en/api) - Free tier, no API key required
