# Spark Protocol

Spark Protocol is a DeFi lending protocol built on top of MakerDAO/Sky. It allows users to deposit liquid staking tokens (LSTs like stETH) as collateral to borrow DAI. SPK is the upcoming governance token.

## Actions

- **Get Protocol TVL** - Fetch the current total value locked in Spark Protocol via DeFiLlama
- **Get DAI Price** - Fetch current DAI price and market data from CoinGecko
- **Get Chain Breakdown** - Get TVL breakdown by blockchain for Spark Protocol
- **Get TVL History** - Fetch last 30 days of historical TVL data for Spark Protocol
- **Get Protocol Stats** - Get key statistics for Spark Protocol (TVL, chains, category)

## APIs Used

- [DeFiLlama API](https://defillama.com/docs/api) - Free, no auth required
- [CoinGecko API](https://www.coingecko.com/en/api) - Free tier, no auth required
