# Pieces Documentation & Samples

This folder contains documentation, samples, and reference materials for building Activepieces pieces.

## Folder Structure

- **documentation/** - API docs, integration guides, and technical documentation
- **samples/** - Sample code, example implementations, and snippets
- **templates/** - Boilerplate templates for creating new pieces
- **references/** - Reference materials, specs, and other useful resources

## Usage

Use this folder to organize any materials you need while developing pieces, such as:
- API documentation from third-party services
- Example requests/responses
- Authentication flow diagrams
- Test data and mock responses
- Code snippets and patterns
- Notes and planning documents
